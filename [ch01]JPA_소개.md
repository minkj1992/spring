# JPA 소개

## SQL 중심적인 개발의 문제점
> 객체를 DB에 넣기 위해서는 SQL을 거칠 수 밖에 없다.
- 무한 반복, 지루한 코드
- 패러다임의 불일치(객체 vs 관계형 DB)
- 엔티티 신뢰문제

## JPA?
> Java Persistence API
- 자바 진영의 ORM 기술 표준
- `EJB -> Hibernate(오픈소스) -> JPA(자바 표준)`
- 캐싱/버퍼링
- 지연 로딩/ 즉시 로딩
    - 지연로딩은 필요한 시점에 불러온다.
        - query 양이 많아진다.
        - 그리하여 상관관계가 큰 table끼리는 option을 주어 즉시로딩하여, join시킨다.
